Variable

trait Variable[A, DomainT[A] <: Domain[A]] extends AnyRef

The Variable type defines the API contract all types representing a com.tubros.constraints.api.solver.Domain-constrained symbolic variable must satisfy in order to be able to minimally participate in the evaluation of a com.tubros.constraints.api.solver.Solver. This largely concerns identification of which variable an instance represents (provided by its name) and what com.tubros.constraints.api.solver.Domain of values it can take on.

To assist in for comprehension use, Variable is domain-leaning in its map' operation. However, situations which call for mapping with both the com.tubros.constraints.api.VariableName and com.tubros.constraints.api.solver.Domain can use the withName method.
