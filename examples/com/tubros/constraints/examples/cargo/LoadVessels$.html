LoadVessels

object LoadVessels extends App with Timer

The LoadVessels type is a scala.App which serves as the entry-point for the a simple Bin packaging problem example program.

There's not much error checking done on the command line arguments, as this is not a production-ready example. Instead, the intent is to solve a real-world problem without introducing logic which a production system would need to have. Ideally, this will lead to a more discoverable example.
